# SSPanel-UIM 宝塔面板安装指南

> **本指南针对已创建网站目录的情况**
> 
> **网站目录**: `/www/wwwroot/jp.moneyfly.dpdns.org`  
> **域名**: `jp.moneyfly.dpdns.org`

## 📋 前提条件

- ✅ 已安装宝塔面板（BT Panel）
- ✅ 已登录宝塔面板
- ✅ 已创建网站目录：`/www/wwwroot/jp.moneyfly.dpdns.org`
- ✅ 代码已上传到 GitHub（仓库：`moneyfly1/myweb`）
- ✅ 域名：`jp.moneyfly.dpdns.org`

## 🚀 快速开始（针对您的目录）

### 一键安装命令

如果您已经安装了 PHP、MariaDB、Redis、Nginx，可以直接执行：

```bash
# 进入您已创建的网站目录
cd /www/wwwroot/jp.moneyfly.dpdns.org

# 下载并运行安装脚本
wget https://raw.githubusercontent.com/moneyfly1/myweb/main/install.sh
chmod +x install.sh
sudo ./install.sh
```

**安装过程中会询问**：
- 域名：输入 `jp.moneyfly.dpdns.org`
- 数据库 root 密码：输入您在宝塔面板中设置的密码
- 数据库名称：默认 `sspanel`（可自定义）
- 数据库用户名：默认 `sspanel`（可自定义）
- 数据库密码：设置强密码
- 管理员邮箱：用于登录管理后台
- 管理员密码：设置管理员登录密码
- 是否配置 SSL：选择 `n`（稍后在宝塔面板中手动配置）

**安装完成后**：
1. 在宝塔面板中添加网站：
   - 域名：`jp.moneyfly.dpdns.org`
   - 根目录：`/www/wwwroot/jp.moneyfly.dpdns.org/public`
   - PHP版本：选择 PHP 8.2 或更高版本
2. 配置伪静态规则（见下方详细说明）
3. 在宝塔面板中申请 SSL 证书

## 🚀 方法一：使用安装脚本（推荐）

安装脚本会自动检测宝塔面板环境并适配配置。

> **重要**：请确保您已经在宝塔面板中创建了网站目录 `/www/wwwroot/jp.moneyfly.dpdns.org`

### 步骤 1: 在宝塔面板中安装基础软件

#### 1.1 安装 PHP

1. 登录宝塔面板
2. 进入 **软件商店** → **运行环境**
3. 安装 **PHP 8.2** 或更高版本（推荐 PHP 8.4）
4. 安装后，点击 **设置** → **安装扩展**，安装以下扩展：
   - ✅ bcmath
   - ✅ bz2
   - ✅ curl
   - ✅ fileinfo（通常已内置）
   - ✅ gd
   - ✅ gmp
   - ✅ intl
   - ✅ mbstring
   - ✅ mysqli
   - ✅ opcache（强烈推荐）
   - ✅ redis
   - ✅ soap
   - ✅ xml
   - ✅ yaml
   - ✅ zip

#### 1.2 安装 MySQL/MariaDB

1. 进入 **软件商店** → **数据库**
2. 安装 **MariaDB 10.11+** 或 **MySQL 8.0+**（推荐 MariaDB 11.8）
3. 记录数据库 root 密码（安装时会显示）

#### 1.3 安装 Redis

1. 进入 **软件商店** → **运行环境**
2. 安装 **Redis 7.0+**
3. 安装后，点击 **设置** → **性能调整**，根据需要调整配置

#### 1.4 安装 Nginx

1. 进入 **软件商店** → **运行环境**
2. 安装 **Nginx**（如果未安装）

#### 1.5 安装 Git（如果未安装）

在宝塔面板的 **终端** 中执行：

```bash
# Debian/Ubuntu
apt install -y git

# CentOS/RHEL
yum install -y git
```

### 步骤 2: 下载并运行安装脚本

#### 2.1 通过宝塔终端下载安装脚本

1. 登录宝塔面板
2. 进入 **终端**
3. 执行以下命令：

```bash
# 进入您已创建的网站目录
cd /www/wwwroot/jp.moneyfly.dpdns.org

# 下载安装脚本
wget https://raw.githubusercontent.com/moneyfly1/myweb/main/install.sh
chmod +x install.sh

# 运行安装脚本（脚本会在当前目录安装）
sudo ./install.sh
```

**重要提示**：
- 安装脚本会在**当前目录**安装项目
- 确保您已经 `cd` 到 `/www/wwwroot/jp.moneyfly.dpdns.org` 目录
- 如果目录不存在，请先在宝塔面板中创建网站，或手动创建：
  ```bash
  mkdir -p /www/wwwroot/jp.moneyfly.dpdns.org
  cd /www/wwwroot/jp.moneyfly.dpdns.org
  ```

#### 2.2 安装脚本会自动执行

安装脚本会：
- ✅ 检测到宝塔面板环境
- ✅ 自动适配宝塔面板的 PHP、MySQL、Redis 路径
- ✅ 从 GitHub 克隆代码到 `/www/wwwroot/jp.moneyfly.dpdns.org`
- ✅ 安装 Composer 依赖
- ✅ 配置数据库
- ✅ 创建 Nginx 配置文件（在 `/www/server/panel/vhost/nginx/` 目录）
- ✅ 初始化数据库
- ✅ 创建管理员账号

**安装过程中会询问**：
- 域名：输入 `jp.moneyfly.dpdns.org`
- 数据库 root 密码：输入您在宝塔面板中设置的密码
- 数据库名称、用户名、密码：可以自定义或使用默认值
- 管理员邮箱和密码：用于登录管理后台

### 步骤 3: 在宝塔面板中配置网站

#### 3.1 添加网站

**如果网站已存在**：
1. 登录宝塔面板
2. 进入 **网站** → 找到 `jp.moneyfly.dpdns.org` → 点击 **设置**
3. 修改 **网站目录** → **根目录** 为：`/www/wwwroot/jp.moneyfly.dpdns.org/public`
4. 点击 **保存**

**如果网站不存在**：
1. 登录宝塔面板
2. 进入 **网站** → **添加站点**
3. 填写信息：
   - **域名**: `jp.moneyfly.dpdns.org`
   - **备注**: SSPanel-UIM
   - **根目录**: `/www/wwwroot/jp.moneyfly.dpdns.org/public`（**重要：必须指向 public 目录**）
   - **FTP**: 不创建
   - **数据库**: 不创建（安装脚本已创建）
   - **PHP版本**: 选择 PHP 8.2 或更高版本
4. 点击 **提交**

#### 3.2 配置网站设置

1. 点击网站右侧的 **设置**
2. **网站目录**：
   - 运行目录：`/public`
   - 防跨站攻击：关闭（或添加项目目录到排除列表）
3. **伪静态**：
   - 选择 **thinkphp** 或手动添加以下规则：

```nginx
location / {
    try_files $uri $uri/ /index.php?$query_string;
}
```

4. **PHP版本**：确保选择 PHP 8.2 或更高版本
5. 点击 **保存**

#### 3.3 配置 SSL 证书（推荐）

1. 在网站设置中，点击 **SSL**
2. 选择 **Let's Encrypt**
3. 填写域名，点击 **申请**
4. 申请成功后，开启 **强制 HTTPS**

## 🔧 方法二：完全手动安装

如果您想完全通过宝塔面板手动安装，可以按照以下步骤：

### 步骤 1: 在宝塔面板中安装软件

按照 **方法一** 的步骤 1 安装所有必需软件。

### 步骤 2: 从 GitHub 克隆代码

在宝塔面板的 **终端** 中执行：

```bash
# 进入您已创建的网站目录
cd /www/wwwroot/jp.moneyfly.dpdns.org

# 克隆代码（直接克隆到当前目录）
git clone https://github.com/moneyfly1/myweb.git .

# 如果遇到 "dubious ownership" 错误
git config --global --add safe.directory /www/wwwroot/jp.moneyfly.dpdns.org
```

### 步骤 3: 安装 Composer 依赖

```bash
# 安装 Composer（如果未安装）
curl -sS https://getcomposer.org/installer | php
mv composer.phar /usr/local/bin/composer
chmod +x /usr/local/bin/composer

# 进入项目目录
cd /www/wwwroot/jp.moneyfly.dpdns.org

# 安装依赖
composer install --no-dev --optimize-autoloader
```

### 步骤 4: 配置数据库

#### 4.1 在宝塔面板中创建数据库

1. 进入 **数据库** → **添加数据库**
2. 填写信息：
   - **数据库名**: `sspanel`
   - **用户名**: `sspanel`
   - **密码**: 设置强密码（请记录）
   - **访问权限**: 本地服务器
3. 点击 **提交**

#### 4.2 配置数据库连接

```bash
# 复制配置文件模板
cd /www/wwwroot/jp.moneyfly.dpdns.org
cp config/.config.example.php config/.config.php

# 编辑配置文件（使用宝塔文件管理器或 nano）
nano config/.config.php
```

修改以下配置：

```php
// 数据库配置
$_ENV['db_host'] = '127.0.0.1';
$_ENV['db_database'] = 'sspanel';
$_ENV['db_username'] = 'sspanel';
$_ENV['db_password'] = '你的数据库密码';
$_ENV['db_port'] = '3306';

// 站点配置
$_ENV['baseUrl'] = 'https://jp.moneyfly.dpdns.org';  // 您的域名
$_ENV['key'] = 'ChangeMe';  // 修改为随机字符串
$_ENV['muKey'] = 'ChangeMe';  // 修改为随机字符串

// Redis 配置（宝塔面板默认配置）
$_ENV['redis_host'] = '127.0.0.1';
$_ENV['redis_port'] = 6379;
$_ENV['redis_password'] = '';  // 如果 Redis 设置了密码，填写密码
```

### 步骤 5: 初始化数据库

```bash
cd /www/wwwroot/jp.moneyfly.dpdns.org

# 运行数据库迁移
php xcat Migration new

# 导入配置项
php xcat Tool importSetting
```

### 步骤 6: 创建管理员账号

```bash
cd /www/wwwroot/jp.moneyfly.dpdns.org

# 创建管理员
php xcat Tool createAdmin
```

按照提示输入：
- 管理员邮箱
- 管理员密码

### 步骤 7: 配置网站（同方法一的步骤 3）

按照 **方法一** 的步骤 3 配置网站。

### 步骤 8: 配置定时任务

在宝塔面板中：

1. 进入 **计划任务** → **添加计划任务**
2. 任务类型：**Shell 脚本**
3. 任务名称：`SSPanel 定时任务`
4. 执行周期：**N分钟**，设置为 `5`
5. 脚本内容：

```bash
/usr/bin/php /www/wwwroot/jp.moneyfly.dpdns.org/xcat Cron
```

6. 点击 **添加任务**

## 🔍 宝塔面板特定配置

### 1. 文件权限设置

在宝塔面板的 **终端** 中执行：

```bash
cd /www/wwwroot/jp.moneyfly.dpdns.org

# 设置目录权限
chmod -R 755 storage
chmod -R 755 config
chown -R www:www storage
chown -R www:www config
chown -R www:www /www/wwwroot/jp.moneyfly.dpdns.org
```

### 2. PHP 配置优化

1. 进入 **软件商店** → **PHP 8.2**（或您使用的版本）→ **设置**
2. **配置修改** → 找到 `php.ini`
3. 修改以下配置：

```ini
memory_limit = 256M
max_execution_time = 300
upload_max_filesize = 50M
post_max_size = 50M
max_input_vars = 5000

; OPcache 配置（如果未启用）
opcache.enable = 1
opcache.memory_consumption = 128
opcache.interned_strings_buffer = 8
opcache.max_accelerated_files = 10000
opcache.revalidate_freq = 2
opcache.fast_shutdown = 1
```

4. 点击 **保存**，然后 **重启 PHP-FPM**

### 3. 禁用 MySQL 严格模式

1. 进入 **软件商店** → **MariaDB**（或 MySQL）→ **设置**
2. **配置修改** → 找到 `my.cnf`
3. 在 `[mysqld]` 部分添加：

```ini
sql_mode = ""
```

4. 点击 **保存**，然后 **重启 MySQL**

### 4. Redis 配置（如果需要密码）

1. 进入 **软件商店** → **Redis** → **设置**
2. **配置修改** → 找到 `redis.conf`
3. 取消注释并设置密码：

```
requirepass your_redis_password
```

4. 在项目配置文件 `config/.config.php` 中更新：

```php
$_ENV['redis_password'] = 'your_redis_password';
```

5. 点击 **保存**，然后 **重启 Redis**

## 🐛 常见问题排查

### 问题 1: 网站显示 403 或 500 错误

**解决方案**:

1. 检查网站根目录是否正确指向 `/www/wwwroot/jp.moneyfly.dpdns.org/public`
2. 检查文件权限：
   ```bash
   chmod -R 755 /www/wwwroot/jp.moneyfly.dpdns.org
   chown -R www:www /www/wwwroot/jp.moneyfly.dpdns.org
   ```
3. 检查 PHP 版本是否正确
4. 查看错误日志：**网站** → **设置** → **日志**

### 问题 2: 数据库连接失败

**解决方案**:

1. 检查数据库服务是否运行：**软件商店** → **MariaDB** → **设置** → **服务**
2. 检查数据库配置是否正确：`config/.config.php`
3. 测试数据库连接：
   ```bash
   mysql -u sspanel -p sspanel
   ```

### 问题 3: Composer 安装失败

**解决方案**:

1. 检查 PHP 扩展是否全部安装
2. 检查 putenv 函数是否被禁用：
   ```bash
   php -r "var_dump(function_exists('putenv'));"
   ```
3. 如果被禁用，在 PHP 配置中移除 `putenv` 从 `disable_functions`

### 问题 4: 定时任务不执行

**解决方案**:

1. 检查计划任务是否添加成功：**计划任务** → 查看任务列表
2. 手动测试执行：
   ```bash
   /usr/bin/php /www/wwwroot/jp.moneyfly.dpdns.org/xcat Cron
   ```
3. 查看执行日志：**计划任务** → 点击任务 → **日志**

### 问题 5: Redis 连接失败

**解决方案**:

1. 检查 Redis 服务是否运行：**软件商店** → **Redis** → **设置** → **服务**
2. 检查 Redis 密码配置是否正确
3. 测试 Redis 连接：
   ```bash
   redis-cli ping
   # 如果有密码
   redis-cli -a your_password ping
   ```
