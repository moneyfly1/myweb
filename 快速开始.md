# ⚡ 快速开始指南

## 🎯 两个步骤完成所有操作

### 步骤 1: 同步代码到 GitHub（在 Mac 上）

```bash
cd /Users/apple/Downloads/SSPanel-UIM-master
./sync_to_github.sh
```

**就这么简单！** 脚本会自动完成所有同步工作。

---

### 步骤 2: 在 VPS 上一键安装（SSH 登录后）

```bash
# 下载安装脚本
wget https://raw.githubusercontent.com/moneyfly1/myweb/main/install.sh
chmod +x install.sh

# 运行安装脚本
sudo ./install.sh
```

**按照提示输入：**
- 域名（例如：example.com）
- 是否配置 SSL（推荐选择 Y）

**等待 10-20 分钟，安装完成！**

## 📋 详细步骤

### 一、同步到 GitHub

#### 使用脚本（推荐）

```bash
cd /Users/apple/Downloads/SSPanel-UIM-master
./sync_to_github.sh
```

#### 手动操作

```bash
cd /Users/apple/Downloads/SSPanel-UIM-master

git init
git remote add origin https://github.com/moneyfly1/myweb.git
git add .
git commit -m "feat: Add node collector feature"
git branch -M main
git push -u origin main --force
```

---

### 二、VPS 安装

#### 1. SSH 登录 VPS

```bash
ssh root@your-vps-ip
# 输入密码
```

#### 2. 下载安装脚本

```bash
wget https://raw.githubusercontent.com/moneyfly1/myweb/main/install.sh
chmod +x install.sh
```

#### 3. 运行安装脚本

```bash
sudo ./install.sh
```

#### 4. 输入信息

```
请输入您的域名 (例如: example.com): your-domain.com
是否现在配置 SSL 证书? (Y/n): Y
```

#### 5. 等待安装完成

脚本会自动：
- ✅ 安装所有软件（Nginx, PHP, MariaDB, Redis）
- ✅ 从 GitHub 克隆项目
- ✅ 配置数据库
- ✅ 配置 Nginx
- ✅ 初始化数据库
- ✅ 创建管理员账户
- ✅ 配置定时任务
- ✅ 配置 SSL 证书

#### 6. 保存重要信息

安装完成后会显示：
- 🔑 数据库密码
- 🔑 Cookie 密钥
- 🔑 WebAPI 密钥

**请务必保存！**

#### 7. 访问网站

- 🌐 网站: `https://your-domain.com`
- 🔐 管理后台: `https://your-domain.com/auth/login`
- 📊 节点采集: `https://your-domain.com/admin/node/collector`

---

## 🎯 配置节点采集

1. 登录管理后台
2. 进入 **节点管理** → **节点采集**
3. 添加采集源 URL
4. 点击 **保存配置**
5. 点击 **立即采集** 测试

---

## ⚠️ 注意事项

1. **同步到 GitHub 会删除原代码** - 请确认
2. **VPS 需要至少 1GB RAM** - 确保资源充足
3. **域名需要解析到 VPS** - 安装前配置好 DNS
4. **保存所有密码和密钥** - 安装后无法找回

## 🆘 遇到问题？

### 安装脚本失败

```bash
# 查看错误信息
# 检查网络连接
ping github.com

# 检查系统版本
cat /etc/os-release
```

### 网站无法访问

```bash
# 检查服务
systemctl status nginx
systemctl status php8.2-fpm
systemctl status mariadb

# 查看日志
tail -f /var/log/nginx/sspanel_error.log
```
