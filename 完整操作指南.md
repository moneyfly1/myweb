## 第二部分：在 VPS 上一键安装

### 📋 前置要求

- VPS 服务器（Debian 12 / Ubuntu 20.04+ / CentOS 8+）
- Root 权限
- 域名（可选，但推荐）

### 🚀 一键安装步骤

#### 步骤 1: SSH 登录 VPS

```bash
ssh root@your-vps-ip
# 输入密码
```

#### 步骤 2: 下载并运行安装脚本

```bash
# 下载安装脚本
wget https://raw.githubusercontent.com/moneyfly1/myweb/main/install.sh

# 或者如果 wget 不可用，使用 curl
# curl -O https://raw.githubusercontent.com/moneyfly1/myweb/main/install.sh

# 添加执行权限
chmod +x install.sh

# 运行安装脚本
sudo ./install.sh
```

#### 步骤 3: 按照提示操作

安装脚本会提示您输入：

1. **域名**（例如：example.com）
   ```
   请输入您的域名 (例如: example.com): example.com
   ```

2. **SSL 证书配置**（推荐选择 Y）
   ```
   是否现在配置 SSL 证书? (Y/n): Y
   ```

#### 步骤 4: 等待安装完成

安装脚本会自动完成：
- ✅ 安装 Nginx、PHP 8.2+、MariaDB、Redis
- ✅ 安装 Composer
- ✅ 从 GitHub 克隆项目
- ✅ 安装 PHP 依赖
- ✅ 配置数据库（自动生成密码）
- ✅ 配置项目文件
- ✅ 设置文件权限
- ✅ 配置 Nginx
- ✅ 初始化数据库
- ✅ 创建管理员账户
- ✅ 配置定时任务
- ✅ 配置 SSL 证书（如果选择）

#### 步骤 5: 保存重要信息

安装完成后，脚本会显示：
- 🔑 **数据库密码** - 请保存
- 🔑 **Cookie 加密密钥** - 请保存
- 🔑 **WebAPI 密钥** - 请保存

**请务必保存这些信息！**

#### 步骤 6: 访问网站

安装完成后，访问：
- 🌐 **网站地址**: `https://your-domain.com`
- 🔐 **管理后台**: `https://your-domain.com/auth/login`
- 📊 **节点采集**: `https://your-domain.com/admin/node/collector`

## 第三部分：配置节点采集功能

### 步骤 1: 登录管理后台

1. 访问 `https://your-domain.com/auth/login`
2. 使用安装时创建的管理员账户登录

### 步骤 2: 配置节点采集

1. 进入 **节点管理** → **节点采集**
2. 在"采集源 URL 列表"中添加节点源 URL（每行一个）
   ```
   https://example.com/subscribe1
   https://example.com/subscribe2
   ```
3. 设置"更新间隔"（建议 3600 秒，即 1 小时）
4. 启用"定时采集"（可选）
5. 添加"过滤关键词"（可选，包含这些关键词的节点将被过滤）
6. 点击 **保存配置**

### 步骤 3: 测试采集

1. 点击 **立即采集** 按钮
2. 等待采集完成（查看日志）
3. 返回 **节点管理** → **节点列表** 查看采集到的节点

## 🔧 安装脚本功能说明

`install.sh` 脚本会自动完成：

### 1. 系统检测
- 自动检测 Debian/Ubuntu/CentOS
- 根据系统选择相应的安装命令

### 2. 安装软件
- **Nginx** - Web 服务器
- **PHP 8.2+** - PHP 运行环境（包含所有必需扩展）
- **MariaDB 11.8** - 数据库（推荐 LTS 版本）
- **Redis 7.0+** - 缓存服务
- **Composer** - PHP 依赖管理

### 3. 配置数据库
- 自动创建数据库 `sspanel`
- 自动创建用户 `sspanel`
- 自动生成强密码

### 4. 部署项目
- 从 GitHub 克隆项目
- 安装 Composer 依赖
- 复制配置文件

### 5. 自动配置
- 生成 Cookie 加密密钥
- 生成 WebAPI 密钥
- 配置数据库连接
- 配置域名

### 6. 设置权限
- 设置正确的文件权限
- 确保 storage 目录可写

### 7. 配置 Nginx
- 创建 Nginx 配置文件
- 配置 PHP-FPM
- 配置 SSL（如果选择）

### 8. 初始化数据库
- 运行数据库迁移
- 导入配置项
- 创建管理员账户

### 9. 配置定时任务
- 添加 Cron 任务（每 5 分钟执行一次）

## 🐛 故障排查

### 安装脚本失败

```bash
# 查看错误信息
# 脚本会显示具体错误

# 常见问题：
# 1. 网络问题 - 检查网络连接
# 2. 权限问题 - 确保使用 root 用户
# 3. 系统不支持 - 检查系统版本
```

### 网站无法访问

```bash
# 检查服务状态
systemctl status nginx
systemctl status php8.2-fpm  # 或 php-fpm
systemctl status mariadb
systemctl status redis

# 检查 Nginx 配置
nginx -t

# 查看错误日志
tail -f /var/log/nginx/sspanel_error.log
```

### 数据库连接失败

```bash
# 检查数据库服务
systemctl status mariadb

# 检查配置文件
cat /var/www/sspanel/config/.config.php | grep db_

# 测试数据库连接
mysql -u sspanel -p -h 127.0.0.1 sspanel
```
