# 数据库问题解决指南

## 问题描述

创建管理员账号时出现 "Database Error" 错误。

## 快速修复步骤

### 步骤 1: 检查数据库配置

```bash
# 进入项目目录
cd /www/wwwroot/board.moneyfly.club

# 下载并运行数据库检查脚本
wget https://raw.githubusercontent.com/moneyfly1/myweb/main/check_database.sh
chmod +x check_database.sh
sudo ./check_database.sh
```

### 步骤 2: 检查数据库服务

```bash
# 检查数据库服务是否运行
systemctl status mariadb
# 或
systemctl status mysql

# 如果未运行，启动它
systemctl start mariadb
# 或
systemctl start mysql
```

### 步骤 3: 检查数据库配置

```bash
# 查看配置文件
cat /www/wwwroot/board.moneyfly.club/config/.config.php | grep -E "db_"
```

应该看到类似：
```php
'db_host' => 'localhost',
'db_database' => 'sspanel',
'db_username' => 'sspanel',
'db_password' => 'your_password',
```

### 步骤 4: 测试数据库连接

```bash
# 读取数据库密码
DB_PASS=$(grep "db_password" /www/wwwroot/board.moneyfly.club/config/.config.php | sed "s/.*=>\s*['\"]\(.*\)['\"].*/\1/" | tr -d "',\"")

# 测试连接
mysql -u sspanel -p"$DB_PASS" sspanel -e "SELECT 1;"
```

如果连接失败，检查：
1. 数据库用户是否存在
2. 密码是否正确
3. 数据库是否存在

## 手动修复方法

### 1. 检查数据库是否存在

```bash
mysql -u root -p
```

在 MySQL 中执行：
```sql
SHOW DATABASES;
USE sspanel;
SHOW TABLES;
```

### 2. 如果数据库不存在，创建它

```bash
mysql -u root -p <<EOF
CREATE DATABASE IF NOT EXISTS sspanel CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
CREATE USER IF NOT EXISTS 'sspanel'@'localhost' IDENTIFIED BY 'your_password';
GRANT ALL PRIVILEGES ON sspanel.* TO 'sspanel'@'localhost';
FLUSH PRIVILEGES;
EOF
```

### 3. 运行数据库迁移

```bash
cd /www/wwwroot/board.moneyfly.club

# 运行迁移
php xcat Migration new
php xcat Migration latest
php xcat Tool importSetting
```

### 4. 创建管理员账号

```bash
php xcat Tool createAdmin
```

## 常见问题

### Q: 数据库连接失败

**A:** 检查以下几点：

1. **数据库服务是否运行**:
   ```bash
   systemctl status mariadb
   ```

2. **数据库用户和密码**:
   ```bash
   mysql -u root -p
   SELECT User, Host FROM mysql.user;
   ```

3. **配置文件中的密码是否正确**:
   ```bash
   cat config/.config.php | grep db_password
   ```

### Q: 数据库表不存在

**A:** 运行数据库迁移：

```bash
cd /www/wwwroot/board.moneyfly.club
php xcat Migration new
php xcat Migration latest
php xcat Tool importSetting
```

### Q: 忘记数据库密码

**A:** 重置数据库密码：

```bash
# 停止数据库服务
systemctl stop mariadb

# 以安全模式启动
mysqld_safe --skip-grant-tables &

# 连接并重置密码
mysql -u root
```

在 MySQL 中：
```sql
USE mysql;
UPDATE user SET password=PASSWORD('new_password') WHERE User='sspanel';
FLUSH PRIVILEGES;
EXIT;
```

然后重启数据库服务。

### Q: 配置文件格式错误

**A:** 检查配置文件语法：

```bash
php -r "require 'config/.config.php';"
```

如果有错误，修复配置文件。

## 验证修复

修复完成后，验证：

```bash
# 1. 测试数据库连接
cd /www/wwwroot/board.moneyfly.club
php -r "
require 'vendor/autoload.php';
\$config = require 'config/.config.php';
echo 'Database: ' . \$config['db_database'] . PHP_EOL;
"

# 2. 检查数据库表
mysql -u sspanel -p sspanel -e "SHOW TABLES;"

# 3. 创建管理员账号
php xcat Tool createAdmin
```

## 如果仍然失败

1. **查看详细错误信息**:
   ```bash
   php xcat Tool createAdmin 2>&1
   ```

2. **检查 PHP 错误日志**:
   ```bash
   tail -50 /var/log/php8.2-fpm.log
   # 或
   tail -50 /www/server/php/82/var/log/php-fpm.log
   ```

3. **检查数据库日志**:
   ```bash
   tail -50 /var/log/mysql/error.log
   ```
