# VPS 快速安装命令

## 方法 1: 一键安装到 /www/wwwroot/board.moneyfly.club（推荐）

在 VPS 上执行：

```bash
bash <(curl -sSL https://raw.githubusercontent.com/moneyfly1/myweb/main/一键安装.sh)
```

选择选项 1，输入域名 board.moneyfly.club

## 方法 2: 直接指定目录安装

```bash
wget https://raw.githubusercontent.com/moneyfly1/myweb/main/install.sh
chmod +x install.sh
export INSTALL_DIR="/www/wwwroot/board.moneyfly.club"
export DOMAIN="board.moneyfly.club"
./install.sh
```

## 方法 3: 交互式安装

```bash
wget https://raw.githubusercontent.com/moneyfly1/myweb/main/install.sh
chmod +x install.sh
./install.sh
```

安装过程中选择：
- 安装目录：选择 2（/www/wwwroot/board.moneyfly.club）
- 域名：输入 board.moneyfly.club

## 安装完成后

### 1. 创建管理员账号

```bash
cd /www/wwwroot/board.moneyfly.club
wget https://raw.githubusercontent.com/moneyfly1/myweb/main/create_admin.sh
chmod +x create_admin.sh
./create_admin.sh
```

### 2. 访问后台

- 前台: http://board.moneyfly.club
- 后台: http://board.moneyfly.club/auth/login
