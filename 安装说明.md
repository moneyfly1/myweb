# 安装说明

## 快速安装

### 方法 1: 在目标目录直接安装（推荐）

```bash
# 1. 进入目标目录
cd /www/wwwroot/board.moneyfly.club

# 2. 下载并运行安装脚本
wget https://raw.githubusercontent.com/moneyfly1/myweb/main/install.sh
chmod +x install.sh
./install.sh
```

脚本会自动检测当前目录，直接在当前目录安装。

### 方法 2: 使用一键安装脚本

```bash
# 在任何目录运行
bash <(curl -sSL https://raw.githubusercontent.com/moneyfly1/myweb/main/一键安装.sh)
```

### 方法 3: 指定目录安装

```bash
# 下载安装脚本
wget https://raw.githubusercontent.com/moneyfly1/myweb/main/install.sh
chmod +x install.sh

# 进入目标目录
cd /www/wwwroot/board.moneyfly.club

# 运行安装脚本（会自动使用当前目录）
./install.sh
```

## 安装流程

安装脚本会自动完成以下步骤：

1. **检测系统环境** - 自动检测操作系统
2. **安装基础工具** - curl, wget, git 等
3. **安装/检测 Nginx** - 如果已安装则跳过
4. **安装/检测 PHP 8.2+** - 自动检测并安装扩展
5. **安装/检测 MariaDB** - 如果已安装则跳过
6. **安装/检测 Redis** - 如果已安装则跳过
7. **安装/检测 Composer** - 自动升级到 2.1+
8. **下载项目代码** - 下载到当前目录或指定目录
9. **安装依赖** - 自动安装 Composer 依赖
10. **设置文件权限** - 自动创建目录并设置权限
11. **配置 Nginx** - 自动配置虚拟主机
12. **初始化数据库** - 运行数据库迁移
13. **创建管理员账号** - 交互式输入邮箱和密码
14. **配置定时任务** - 自动配置 Cron
15. **配置 SSL 证书** - 可选，使用 Certbot

## 安装目录说明

- **自动检测**: 脚本会根据当前工作目录自动确定安装位置
- **当前目录**: 如果当前目录看起来像项目目录，直接使用
- **手动选择**: 也可以手动选择或输入安装目录

## 使用示例

### 示例 1: 在宝塔面板目录安装

```bash
# 进入宝塔面板网站目录
cd /www/wwwroot/board.moneyfly.club

# 运行安装脚本
wget https://raw.githubusercontent.com/moneyfly1/myweb/main/install.sh
chmod +x install.sh
./install.sh
```

脚本会自动：
- 检测到当前目录是 `/www/wwwroot/board.moneyfly.club`
- 直接在此目录下载和安装项目
- 自动配置 Nginx 指向此目录

### 示例 2: 标准安装

```bash
# 进入标准目录
cd /var/www

# 运行安装脚本
wget https://raw.githubusercontent.com/moneyfly1/myweb/main/install.sh
chmod +x install.sh
./install.sh
```

## 安装后访问

安装完成后，访问：
- **前台**: http://您的域名
- **后台**: http://您的域名/auth/login
- **节点采集**: http://您的域名/admin/node/collector

使用安装过程中创建的管理员账号登录。
