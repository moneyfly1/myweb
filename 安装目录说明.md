# 安装目录选择说明

## 📁 安装脚本的工作原理

**重要**：安装脚本会在**当前工作目录**（`pwd`）安装项目，所以必须先 `cd` 到您想安装的目录，然后再运行安装脚本。

## 🎯 如何选择安装目录

### 1. 宝塔面板用户（推荐）

宝塔面板通常使用 `/www/wwwroot` 作为网站根目录。

```bash
# 进入宝塔网站目录
cd /www/wwwroot

# 创建项目目录
mkdir -p sspanel
cd sspanel

# 下载并运行安装脚本
wget https://raw.githubusercontent.com/moneyfly1/myweb/main/install.sh
chmod +x install.sh
sudo ./install.sh
```

**安装完成后，在宝塔面板中添加网站时，根目录设置为：**
- `/www/wwwroot/sspanel/public` ✅

### 2. 标准 Linux 系统

如果 `/var/www` 不存在，可以使用其他目录：

```bash
# 方式 A: 使用 /opt 目录（推荐）
cd /opt
sudo mkdir -p sspanel
cd sspanel
wget https://raw.githubusercontent.com/moneyfly1/myweb/main/install.sh
chmod +x install.sh
sudo ./install.sh

# 方式 B: 使用 /home/www 目录
cd /home
sudo mkdir -p www/sspanel
cd www/sspanel
wget https://raw.githubusercontent.com/moneyfly1/myweb/main/install.sh
chmod +x install.sh
sudo ./install.sh

# 方式 C: 如果 /var/www 存在
cd /var/www
sudo mkdir -p sspanel
cd sspanel
wget https://raw.githubusercontent.com/moneyfly1/myweb/main/install.sh
chmod +x install.sh
sudo ./install.sh
```

### 3. 测试环境

可以使用当前用户目录：

```bash
# 进入用户目录
cd ~
mkdir sspanel
cd sspanel

# 下载并运行安装脚本
wget https://raw.githubusercontent.com/moneyfly1/myweb/main/install.sh
chmod +x install.sh
sudo ./install.sh
```

## 🔍 如何检查目录是否存在

在运行安装脚本前，可以先检查目录：

```bash
# 检查常见目录
echo "检查 /www/wwwroot:"
ls -d /www/wwwroot 2>/dev/null && echo "✅ 存在" || echo "❌ 不存在"

echo "检查 /var/www:"
ls -d /var/www 2>/dev/null && echo "✅ 存在" || echo "❌ 不存在"

echo "检查 /opt:"
ls -d /opt 2>/dev/null && echo "✅ 存在" || echo "❌ 不存在"

echo "检查 /home/www:"
ls -d /home/www 2>/dev/null && echo "✅ 存在" || echo "❌ 不存在"
```

## 📝 目录选择建议

| 系统类型 | 推荐目录 | 说明 |
|---------|---------|------|
| 宝塔面板 | `/www/wwwroot/sspanel` | 宝塔标准网站目录 |
| Debian/Ubuntu | `/opt/sspanel` | 可选软件安装目录 |
| CentOS/RHEL | `/opt/sspanel` | 可选软件安装目录 |
| 测试环境 | `~/sspanel` | 用户目录，方便测试 |
| 自定义 | `/your/custom/path/sspanel` | 任何有权限的目录 |

## ⚠️ 重要提示

1. **必须先创建目录并进入**：安装脚本会在当前目录安装，所以必须先 `cd` 到目标目录
2. **确保有写入权限**：安装目录需要对运行脚本的用户有写入权限
3. **网站根目录配置**：安装完成后，在 Nginx/宝塔面板中配置网站时，根目录必须指向 `项目目录/public`，例如：
   - 如果安装在 `/www/wwwroot/sspanel`，网站根目录设置为 `/www/wwwroot/sspanel/public`
   - 如果安装在 `/opt/sspanel`，网站根目录设置为 `/opt/sspanel/public`

## 🚀 快速安装命令（宝塔面板）

如果您使用宝塔面板，直接执行：

```bash
cd /www/wwwroot && mkdir -p sspanel && cd sspanel && \
wget https://raw.githubusercontent.com/moneyfly1/myweb/main/install.sh && \
chmod +x install.sh && sudo ./install.sh
```

## 🚀 快速安装命令（标准系统）

如果 `/var/www` 不存在，使用：

```bash
cd /opt && sudo mkdir -p sspanel && cd sspanel && \
wget https://raw.githubusercontent.com/moneyfly1/myweb/main/install.sh && \
chmod +x install.sh && sudo ./install.sh
```
