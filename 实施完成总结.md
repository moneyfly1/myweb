## 🎯 功能特点

1. **自动化采集**
   - 支持从多个 URL 源采集
   - 自动解析多种协议
   - 自动保存到数据库

2. **智能去重**
   - 根据 server + port + sort 判断重复
   - 更新已存在节点，不创建重复节点

3. **灵活配置**
   - 可配置采集间隔
   - 支持关键词过滤
   - 可启用/禁用定时采集

4. **完善日志**
   - 记录采集过程
   - 查看采集历史
   - 错误信息记录

5. **用户友好**
   - 直观的管理界面
   - 实时状态显示
   - 清晰的节点来源标识

## ⚠️ 注意事项

1. **数据库迁移**
   - 必须先运行数据库迁移
   - 迁移文件：`db/migrations/2025010100-add_node_collector.php`

2. **权限要求**
   - 确保 PHP 有网络访问权限（用于下载节点数据）
   - 确保数据库有写入权限

3. **性能考虑**
   - 大量节点采集可能耗时较长
   - 建议采集间隔至少 60 秒
   - 建议在低峰期执行采集

4. **兼容性**
   - 保留手动配置节点功能
   - 采集节点和手动节点可以共存
   - 不影响现有功能
